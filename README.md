# Space Hub

NASA Space Apps Challenge Project  
**Challenge:** Meteor Madness  
**Project Name:** Space Hub

## Overview

Space Hub is an interactive web application for visualizing Near-Earth Objects (NEOs) and simulating asteroid impact scenarios. The platform combines real-time NASA NEO data with physics-based impact simulations and AI-powered analysis.

## Features

- **3D Orrery Visualization**: Interactive 3D visualization of Earth and nearby asteroids using Three.js
- **Real-time NEO Data**: Integration with NASA's NeoWs API for live asteroid tracking
- **Impact Simulation**: Physics-based impact modeling with blast radius calculations
- **Gravity Tractor Analysis**: Deflection mission planning and probability assessment
- **AI-Powered Analysis**: Gemini AI integration for impact consequence modeling
- **Interactive Maps**: MapLibre integration for geographical impact visualization

## Technology Stack

- **Frontend**: Three.js, MapLibre GL, Vanilla JavaScript (ES6 modules)
- **APIs**: NASA NeoWs, NASA Fireball, NASA Meteorite Landings
- **AI**: Google Gemini for impact analysis and formula explanations
- **Deployment**: Vercel (serverless)
